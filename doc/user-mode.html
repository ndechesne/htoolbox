User-mode backup

User-mode backup is a special case of backup. It does not require a server-side configuration, but has some requirements.

Database

The database will be created in the $HOME/.hbackup directory, so you must leave it free for that use.

Configuration file

The configuration file is a standard client-side configuration file, except that paths starting in ~ are allowed, where ~ represents the $HOME path as defined in the current environment.

The configuration file must be $HOME/.hbackup/config.

There is no need for a specific filter to ignore the ~/.hbackup path, as directories named .hbackup are always ignored.
